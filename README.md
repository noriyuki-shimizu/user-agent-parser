# User agent parser

## version

```
node: v12.6.0
npm: 6.9.0
```

## clone

本リポジトリからクローンしてください

## package install

```
$ npm ci
```

## edit main file

`main.ts` ファイルのユーザーエージェント情報を追記・編集してください

```
$ vim src/main.ts
```

## execute

### 実行

```
$ npm run start
```

### 実行結果をファイルに書き出し

```
$ npm run start:output
```
